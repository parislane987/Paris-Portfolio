Desert Bloom Rescue

Desert Bloom Rescue is a group web design project that I created with Samantha Zellhart and Chris Berridge for my web design class. The website provides information about Desert Bloom's mission, animals, team, and volunteering opportunities for animal activists and enthusiasts to explore.
